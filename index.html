Sobre a Calculadora

A Calculadora de Banco de Dados auxilia no dimensionamento de espaço em disco (breve também memória) de um banco MySQL, de acordo com o script de criação do banco (DDL).

É possível também considerar no cálculo, o consumo histórico de espaço usado pelos dados inseridos no banco, caso isto esteja disponível. Caso contrário, somente será calculado o tamanho do banco como se estivesse vazio.

Como o cálculo é feito

A ferramenta lê o script DDL de criação das tabelas e separa os tipos de dados e tamanhos das colunas. Soma a quantidade de bytes correspondente ao tipo de dados de cada coluna, multiplicado pelo tamanho do campo quando aplicável, de acordo com a documentação do MySQL disponível neste link para obter o tamanho do banco vazio.

Caso for informada a estimativa de crescimento de dados armazenados no banco, a ferramenta converte o volume para bytes e multiplica o tamanho do banco vazio.

Ao final efetua as conversões para MB e GB multiplicando pelo período (dia ou mês e ano) conforme selecionado no formulário.

Restrições

  • A Calculadora de Banco de Dados não considera a adição de espaço do índice no cálculo do banco vazio. (Em desenvolvimento)
  • Está sendo considerada nesta versão apenas o banco MySQL 8.0 e o tipo de armazenamento InnoDB.
  • Os tipos de dados ENUM e SET não estão suportados nesta versão da ferramenta.
